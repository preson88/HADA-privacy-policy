데이터 삭제 요청

개발자: HADA 운영팀  ·  문의: preson88@gmail.com

HADA 앱의 모든 사용자 데이터는 기기 로컬에만 저장됩니다. 개발자 자체 서버에는 업로드되지 않습니다. 아래 방법 중 하나를 선택해 데이터를 삭제할 수 있습니다.

방법 1. 앱 내 데이터 초기화

A 레벨 89 이상 사용자
레벨 89 이상 조건 필요

레벨 89에 도달하면 프로필 화면에서 데이터 초기화 기능이 활성화됩니다. 기기에 저장된 모든 데이터가 즉시 삭제되며 복구할 수 없습니다.

앱 실행 → 프로필 → 데이터 초기화
  1. 하단 탭에서 프로필 화면으로 이동합니다.
  2. 데이터 초기화 버튼을 탭합니다.
  3. 확인 모달에서 초기화를 선택합니다.
  4. 앱이 자동으로 온보딩 화면으로 이동하며 삭제가 완료됩니다.
B 레벨 89 미만 사용자
레벨 무관 · 누구나 가능

Android 설정에서 앱 데이터를 직접 삭제하거나, 앱을 제거하면 기기 내 모든 데이터가 삭제됩니다.

Android 설정 → 앱 → HADA → 저장공간 → 데이터 삭제

또는 앱을 길게 눌러 제거해도 동일하게 삭제됩니다.

⚠️ 앱 삭제 후 재설치 시 주의
기기에서 앱을 삭제하면 기기 내 데이터는 삭제됩니다. 단, Android 자동 백업(Google 계정 백업)이 활성화된 경우 재설치 시 일부 데이터가 복원될 수 있습니다.
완전 삭제를 원하시면 앱 삭제 전 아래 설정을 먼저 해제하세요.
Android 설정 → Google → 백업 → 앱 데이터 백업 해제
