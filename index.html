개인정보처리방침

최초 시행일: 2025년 1월 1일  |  최종 수정일: 2025년 4월 15일

본 개인정보처리방침은 앱(이하 "서비스")이 사용자의 개인정보를 어떻게 수집·이용·보관·공유하는지 설명합니다. 서비스를 이용함으로써 본 방침에 동의한 것으로 간주됩니다.

1. 수집하는 개인정보 항목

서비스는 다음의 정보를 수집합니다.

분류 항목 수집 방법
신원 정보 이름, 한자명(선택) 사용자 직접 입력
생체·인적사항 생년월일, 출생 시각, 성별 사용자 직접 입력
위치 관련 출생지 문자열, 위도·경도(온보딩 기본값) 사용자 직접 입력 / 앱 계산
AI 요청 텍스트 운세·행동 제안 등 사용자가 입력한 자연어 질문 사용자 직접 입력
광고 식별자 광고 ID(Android Advertising ID) 기기 자동 수집(AdMob)
기기·네트워크 IP 주소, User-Agent(Google 서버 접속 시) 서비스 이용 자동 수집
※ 습관·목표·할 일·일정·행복 기록·알람 설정 등 앱 내 생성 데이터는 기기 로컬(Capacitor Preferences)에만 저장되며, 개발사 서버로 전송되지 않습니다.

2. 개인정보 수집·이용 목적

  • 사주·자미두수·서양 점성·성명학 분석: 이름·생년월일·출생시각·성별·출생지 정보를 기반으로 운세 정보를 계산합니다.
  • AI 운세 답변 생성: 위 계산 결과 및 사용자 질문 텍스트를 Google Gemini API로 전송하여 자연어 답변을 생성합니다.
  • 맞춤형 광고 제공: Google AdMob을 통해 광고를 게재하며, 이 과정에서 광고 ID가 활용됩니다.
  • 서비스 개선 및 오류 처리: 기기·네트워크 정보를 통해 서비스 품질을 유지합니다.

3. 제3자 제공 및 위탁

서비스는 아래 제3자에게 데이터를 전송합니다. 개발사 자체 백엔드 서버로의 업로드는 없습니다.

제3자 전송 데이터 목적 개인정보처리방침
Google LLC

Google LLC
(AdMob)		 광고 ID, 기기·광고 상호작용 정보, IP 주소 맞춤형 광고 게재·측정·사기 방지 policies.google.com/privacy
Google LLC
(Google Fonts)		 IP 주소, User-Agent 폰트 파일 제공 policies.google.com/privacy

4. 알림(알람) 권한

서비스는 사용자가 설정한 일정·알람을 기기 로컬에서 예약·발송하기 위해 알림 권한을 사용합니다. 알림 관련 데이터는 기기 외부로 전송되지 않으며, 알람 설정은 기기 내 로컬 저장소에만 보관됩니다.

5. 개인정보 보유 및 파기

  • 로컬 데이터: 사용자가 앱을 삭제하거나 앱 데이터를 초기화하는 즉시 삭제됩니다.
  • Google AI·광고 서버: Google의 데이터 보존 정책에 따르며, 자세한 내용은 Google 개인정보처리방침을 참조하세요.
  • 관련 법령에서 보존을 요구하는 경우 해당 기간 동안 보관 후 파기합니다.

6. 사용자의 권리

  • 열람·수정·삭제 요청: 앱 내 설정에서 직접 수정하거나, 아래 문의처로 삭제 요청할 수 있습니다.
  • 광고 개인화 거부: Android 설정 → Google → 광고 → 광고 개인화 선택 해제로 광고 ID 사용을 제한할 수 있습니다.
  • 미성년자: 본 서비스는 만 14세 미만 아동을 대상으로 하지 않습니다. 만 14세 미만이 수집된 것을 인지한 경우 즉시 삭제 처리합니다.

7. 개인정보 보호 조치

  • 외부로 전송되는 모든 데이터는 HTTPS(TLS) 암호화 통신을 사용합니다.
  • 알람 API 키 등 민감한 설정 값은 기기 로컬 저장소에만 보관합니다.

8. 개인정보처리방침 변경

방침이 변경될 경우 앱 내 공지 또는 본 페이지를 통해 사전 안내합니다. 변경된 방침은 게시 즉시 효력이 발생합니다.
